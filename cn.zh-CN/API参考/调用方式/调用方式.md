# 调用方式 {#concept_29962_zh .concept}

对视频直播 API 的调用是通过向视频直播 API 的服务端地址发送 HTTP GET 请求，并按照接口说明在请求中加入相应请求参数来完成的。根据请求的处理情况，系统会返回处理结果。

-    [公共参数](intl.zh-CN/API参考/调用方式/公共参数.md#)

-    [返回结果](intl.zh-CN/API参考/调用方式/返回结果.md#)
